# 🎓 Student Performance Analyzer

A web-based Student Performance Analyzer developed using **Python Flask**, **SQLite**, **HTML**, **CSS**, and **Matplotlib**.

---

## Features

- 🔐 User Login & Registration
- ➕ Add Student Records
- 📋 View Students
- 🔍 Search Students
- 🗑 Delete Student Records
- 📊 Performance Analysis
- 📈 Student Performance Bar Chart
- 🎯 Automatic Grade Calculation
- 💻 Responsive User Interface

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Matplotlib
